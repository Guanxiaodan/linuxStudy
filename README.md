## ln命令
创建文件链接
### 1.创建文件软连接
> 文件位置：*linuxStudy/ln*

##### *ln*文件夹里有*origin*和*dest*两个文件夹，

```flow
st=>start:Start
e=>end:End
op1=>operation:My Option
sub1=>subroutine:My subroutine
cond=>condition:Yes or No?
io=>inputoutput:catch something
st->op1->cond
cond(yes)->io->e
cond(no)->sub1(right)->op1
```
